# Operators and built-ins

Expressions use built-in functions for numeric operations, string concatenation, selections, properties, page information, and diagnostics. Operator notation is a short form for ordinary function calls. `a + b` calls the same function as `add(a, b)`, and `a ++ b` calls the same function as `concat(a, b)`.

## Functions With Operator Notation

| Function | Operator notation | Function call | Signature |
| --- | --- | --- | --- |
| `neg` | `-x` | `neg(x)` | `number -> number` |
| `add` | `a + b` | `add(a, b)` | `(number, number) -> number` |
| `sub` | `a - b` | `sub(a, b)` | `(number, number) -> number` |
| `mul` | `a * b` | `mul(a, b)` | `(number, number) -> number` |
| `div` | `a / b` | `div(a, b)` | `(number, number) -> number` |
| `concat` | `a ++ b` | `concat(a, b)` | `(string, string) -> string` |

## Examples

```ss
let size = 18 + 4
let label = "page " ++ str(page_index(pagectx()))
```

The same code can be written with ordinary function calls.

```ss
let size = add(18, 4)
let label = concat("page ", str(page_index(pagectx())))
```
