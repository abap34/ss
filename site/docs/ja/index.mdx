# ss

`ss` は，コード，数式，画像，繰り返しの構造を含むスライドを書くための言語とコマンドラインツールです．

デッキは `.ss` ソースとして保存します．共有するコンポーネントは関数にします．ページ番号や目次は `document` ブロックに書きます．配置を固定する箇所には制約を書きます．

次の例は，表紙と本文を持つ最小の資料です．PDF とソースを並べて載せています．
