# 演算子と組み込み関数

式でよく使う組み込み関数には，数値演算，文字列連結，選択，プロパティ操作，ページ情報，診断があります．演算子表記は，普通の関数呼び出しの短い書き方です．`a + b` は `add(a, b)` と同じ関数を呼び，`a ++ b` は `concat(a, b)` と同じ関数を呼びます．関数定義や型注釈は [関数と定義](./functions) を参照してください．値の分類は [値と型](./values-and-types) を参照してください．

## 演算子として書ける関数

演算子として書ける関数は，通常の関数呼び出しとしても書けます．演算子表記では，優先順位と結合規則によって引数のまとまりが決まります．その後は，ほかの関数呼び出しと同じように引数型と返り値型を検査します．

| 関数 | 演算子表記 | 通常の呼び出し | シグネチャ |
| --- | --- | --- | --- |
| `neg` | `-x` | `neg(x)` | `number -> number` |
| `add` | `a + b` | `add(a, b)` | `(number, number) -> number` |
| `sub` | `a - b` | `sub(a, b)` | `(number, number) -> number` |
| `mul` | `a * b` | `mul(a, b)` | `(number, number) -> number` |
| `div` | `a / b` | `div(a, b)` | `(number, number) -> number` |
| `concat` | `a ++ b` | `concat(a, b)` | `(string, string) -> string` |

```ss
let a = 10 + 4
let b = a * 2
let c = -b
let label = "value = " ++ str(c)
```

同じ式を通常の関数呼び出しで書くと次の形になります．

```ss
let a = add(10, 4)
let b = mul(a, 2)
let c = neg(b)
let label = concat("value = ", str(c))
```

演算子の優先順位は [構文](./syntax) の式の節にあります．複数の演算子が混ざる場合は，丸かっこで意図を明示できます．

```ss
let x = 10 + 20 * 3
let y = (10 + 20) * 3
let same_x = add(10, mul(20, 3))
let same_y = mul(add(10, 20), 3)
```

## プロパティ関数

プロパティ参照構文は，内部では `prop`，`has_prop`，`set_prop` に対応します．

<ApiCard
  locale="ja"
  kind="function"
  name="prop"
  signature="prop(target: any, key: string, default: string) -> string"
  module="組み込み"
  returns="string"
  summary="document，page，object のプロパティを読みます．値がない場合は既定値を返します．"
/>

<ApiCard
  locale="ja"
  kind="function"
  name="has_prop"
  signature="has_prop(target: any, key: string) -> bool"
  module="組み込み"
  returns="bool"
  summary="プロパティが存在するかを返します．"
/>

```ss
let body = text("本文")
body.text_size = 22

let size = body.text_size ?? "18"
let exists = body.text_size?
```

この例では，`body.text_size ?? "18"` が `prop(body, "text_size", "18")` と同じ扱いになります．`body.text_size?` は `has_prop(body, "text_size")` と同じ扱いです．プロパティの値と保存形式は [プロパティとスタイル](./properties) を参照してください．

## グラフ生成関数

<ApiCard
  locale="ja"
  kind="function"
  name="new"
  signature="new(page: page, content: string, role_name: string, payload_name: string) -> object"
  module="組み込み"
  returns="object"
  effects="CreateNode，WriteContent"
  summary="指定したページに object を作ります．標準コンポーネントの低水準の入口です．"
/>

```ss
fn raw_label(text_value: string) -> object
  let obj = new(pagectx(), text_value, "label", "text")
  obj.text_size = 14
  return obj
end
```

通常の資料では `new` を直接使うより，`text`，`head`，`image` などのコンポーネント関数を使います．コンポーネントは [コンポーネント](../components/text) を参照してください．

<ApiCard
  locale="ja"
  kind="function"
  name="equal"
  signature="equal(target: anchor, source: anchor, offset: number = 0) -> constraints"
  module="組み込み"
  returns="constraints"
  effects="WriteConstraint"
  summary="二つのアンカーを等式制約で結びます．"
/>

```ss
equal(anchor(body, "top"), anchor(title, "bottom"), -32)
```

この呼び出しは，次の制約文と同じ種類の制約を追加します．

```ss
~ body.top == title.bottom - 32
```

標準ライブラリの `below`，`same_l`，`pin_l` などは `anchor` と `equal` を包む補助関数です．詳しくは [制約](./constraints) と [配置](../layout/default) を参照してください．
